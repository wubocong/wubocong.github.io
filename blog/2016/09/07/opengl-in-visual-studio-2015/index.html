在Visual Studio 2015中配置OpenGL开发环境

2016-09-07 Comments Graphics

参考：SETTING UP OPENGL WITH VISUAL STUDIO

OpenGL有许多工具集，OpenGL官网在这里列出了。它们封装了底层的方法，在国内最常见用的是GLUT，然而这个已经是18年没更新的古董，它的协议不允许他人修改重新发布。我选择了freeglut，它是GLUT的超集也完全兼容旧的代码。在其主页上提供多个版本，官方不提供编译后的二进制文件下载，需要自己编译，在Windows上我使用的是第三方编译版本Martin Payne’s Windows binaries (MSVC and MinGW)。

另外搭配的是GLEW(GL Extension Wrangler)，用来支持最新的OpenGL特性（现在是OpenGL 4.5）。

下载后，将内部的文件按照系统架构对应放到Visual Studio VC对应文件夹下。include文件建议分成两个文件夹，bin文件夹中amd64和x86的顺序和组合的含义是：第一个代表编译平台，第二个代表目标平台。





环境已经配置好，可以开始编码了，打开VS 2015，

选择Visual C++创建空项目并添加main.cpp
